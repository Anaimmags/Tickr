# Tickr

🎟️ TICKR – TICKET ALERT BOT

Automatically finds Ticketmaster resale tickets and reserves them for you.

---

⚡ WHAT IT DOES

• Automatically searches for resale tickets  
• Instantly selects and reserves tickets when available  
• Holds tickets for approximately 1 minute 30 seconds  
• Works directly on Ticketmaster pages  
• Supports multiple tabs (track different ticket types at once)

---

🚀 HOW TO INSTALL

1. Download and unzip the folder
2. Open Google Chrome
3. Go to: chrome://extensions/
4. Enable "Developer Mode" (top right)
5. Click "Load unpacked"
6. Select the unzipped folder

✅ Done! Tickr is now installed

---

🎯 HOW TO USE

1. Go to a Ticketmaster event page
2. Select your ticket type
3. Click "Start bot" in the extension
4. Wait — Tickr will automatically search for tickets

When tickets appear:
👉 They will be automatically reserved for you

---

💎 PREMIUM FEATURES

• 🔊 Sound alert when tickets are found  
• 🖥 Browser tab changes to "TICKETS FOUND"  
• 📲 Telegram alerts sent to your phone  
• ⚡ Faster refresh speed (better chances)

🔐 HOW TO UNLOCK PREMIUM
